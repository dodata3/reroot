typedef struct _QRcode_List QRcode_List

Singly-linked list of QRcode. Used to represent a structured symbols. A list is terminated with NULL.

typedef struct _QRinput QRinput

Singly linked list to contain input strings. An instance of this class contains its version and error correction level too. It is required to set them by QRinput_setVersion() and QRinput_setErrorCorrectionLevel(), or use QRinput_new2() to instantiate an object.
